# 周末行动计划（Weekend Planner）

一个简洁优雅的周末计划网页应用，支持：
- 录入并即时展示行动计划
- 周末时间列表（周五晚上、周六/周日 上午/下午/晚上）
- 餐厅专用时间段（周五/六/日 的中饭/晚饭）
- 拖拽关联计划与时间段
- 本地持久化（localStorage）
- 多设备同步（通过分享链接或 data.json）

> 打开 `https://spring1993.github.io/week_plan/` 即可使用

## 功能一览

- **计划录入**：标题、类型（地点/餐厅/必须完成/事件）、地点/链接、优先级、备注
- **智能时间段**：选择"餐厅"类型时自动切换为用餐时间段
- **拖拽关联**：拖拽计划到具体时间段
- **时间轴表格**：顶部展示所有时间段的完整安排
- **单个清空**：每个时间段可单独清空
- **导出功能**：下载 JSON 文件备份
- **分享功能**：生成带数据的链接，发给别人或其他设备
- **响应式设计**：移动端友好

## 多设备同步方案

由于跨域限制，我们提供两种同步方式：

### 方式一：分享链接（推荐，最简单）
1. 在电脑上修改完计划后，点击右上角"分享"按钮
2. 链接自动复制到剪贴板
3. 把链接发到手机或发给别人
4. 打开链接即可看到相同的计划

### 方式二：通过 GitHub 同步（适合固定设备）
1. 在网页上点击"导出"按钮，下载 `weekend_plans.json`
2. 在项目目录运行同步脚本：
   ```bash
   cd /Users/shijian/.cursor/weekend_plan
   ./sync-to-github.sh
   ```
3. 等待 1-2 分钟 GitHub Pages 部署
4. 其他设备刷新 `https://spring1993.github.io/week_plan/` 即可看到最新数据

## 数据存储

- **本地**：浏览器 localStorage（自动保存，刷新不丢失）
- **云端**：
  - 分享链接：数据编码在 URL 中
  - GitHub 仓库：`data.json` 文件（需手动同步）

## 时间段说明

### 通用时间段（地点/事件/必须完成）
- 周五晚上、周六上午/下午/晚上、周日上午/下午/晚上、随意/暂定

### 餐厅时间段（餐厅类型）
- 周五晚饭、周六中饭/晚饭、周日中饭/晚饭、随意/暂定
