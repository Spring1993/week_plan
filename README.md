# 周末行动计划（Weekend Planner）

一个简洁优雅的周末计划网页应用，支持：
- 录入并即时展示行动计划
- 周末时间列表（周五晚上、周六/周日 上午/下午/晚上）
- 在计划与时间段之间进行拖拽关联
- 本地持久化（localStorage），无需服务器

> 打开 `index.html` 即可使用，数据仅保存在你的浏览器本地。

## 功能一览

- 计划录入：标题、类型（地点/餐厅/必须完成）、地点/链接、优先级、备注
- 可选预分配：在创建计划时即可多选时间段
- 拖拽关联：将左侧待分配计划拖入右侧具体时间段
- 反向操作：从时间段中移除计划，计划会回到待分配列表
- 筛选：按类型过滤待分配计划
- 清空安排：一键清空所有时间段的安排（计划仍保留）
- 导入/导出：备份或迁移到其它设备
- 响应式设计：移动端友好

## 快速开始

1. 打开本项目目录
2. 使用浏览器直接打开 `index.html`
3. 输入你的行动计划，或将示例计划拖进对应时间段

## 技术与设计

- HTML5 语义化结构：`header`/`main`/`footer`
- 样式：内联 CSS，深色主题，卡片式布局，柔和霓虹高亮
- 字体：Google Fonts（Inter、Nunito）
- 图标：Font Awesome CDN
- 交互：原生 Drag & Drop，localStorage 持久化
- 无依赖：无需构建、无需后端

## 数据结构

- `plans`: 数组
  ```json
  {
    "id": "p_xxx",
    "title": "计划标题",
    "type": "place|restaurant|task",
    "location": "可选链接或地点",
    "priority": "normal|high|urgent",
    "notes": "备注"
  }
  ```
- `schedule`: 映射
  ```json
  {
    "fri_pm": ["p_id1", "p_id2"],
    "sat_am": [],
    "sat_pm": [],
    "sat_nt": [],
    "sun_am": [],
    "sun_pm": [],
    "sun_nt": []
  }
  ```

## 使用小贴士

- 可在“新建行动计划”中多选时间段实现预分配
- 任意时刻可通过右上角“导出/导入”备份与恢复
- 删除计划会同时从所有时间段移除

## 可能的改进方向
